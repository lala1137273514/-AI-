## 💡 核心洞察

1. **AI 内部赋能专项从 Kickoff 到对齐会只隔两天，各方向 demo 已有雏形**：何尔宁的"带 demo 的人"定位是对的——不靠概念取胜，靠可演示物推进。第二周工程化落地的难度可能被低估。

2. **旅行规划提示词调优进入深水区**：把 skill 的交互契约翻译成 LLM 提示词，本质是把"设计语言"翻译成"提示词语言"，中间的信息损失和过拟合风险需要逐条验证。本周密集做了 7 个 session 的只读分析+调优，但还没到可验收阶段。

3. **飞书日报 cron 反复掉线是风险**：06-07~06-10 连续四天日报缺失，归档也为空，cron 可能挂了但没告警。日报缺失后，周报的四天只能靠本地扫描和会议纪要硬补，信息密度下降。

4. **Agent UI 协议（AtoUI）是个杠杆点**：agent 直接控制前端显示，往模板槽位填 JSON——产品侧定义 UI 规则时可以直接影响 agent 输出，不再依赖研发翻译。后续旅行规划等复杂链路的 UI 都可复用这套协议。
