## 🛠️ 配色技巧

1. **温度对比** - 冷色环境 + 暖色焦点 = 视觉层次感
2. **7:3比例** - 主色70%，焦点色30%（黄金比例）
3. **单色调+强烈焦点** - 冷色调主体 + 高饱和度暖色焦点
4. **对比度分层** - 低对比主体 + 高对比焦点
5. **特殊效果** - 透明度、光晕、次表面散射增强视觉深度
